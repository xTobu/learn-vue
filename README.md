
# learn-vue

> 根據tpl-vue-simple，並增加axios、qs、element UI、url-loader

 - vue
 - vuex
 - vue-router
 - pug
 - sass-loader
 - axios
 - qs
 - element 
 - url-loader（引用ElementUI樣式時需要編譯字型檔）

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
